Descripción

CMFO-UNIVERSE es la implementación oficial del modelo CMFO en librería de alto rendimiento, incluyendo:

  • Álgebra T⁷
  • Tensores fractales φ
  • Matrices 7×7
  • Kernels CUDA optimizados
  • API C / C++ / Python
  • Pruebas unitarias profesionales

Instalación

Python (próximamente PyPI)

pip install cmfo-universe

C/C++

cmake .. && cmake --build . --config Release
