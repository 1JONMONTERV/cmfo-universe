# Introducción al CMFO

Bienvenido al documento maestro del **CMFO–UNIVERSE v∞**.

Este sistema unifica computación, física, teoría de números y dinámica fractal 7D en un único autómata geométrico basado en el toro fractal \( T^7_\varphi \).

Desde aquí podrás navegar toda la documentación científica, kernels GPU, demostraciones matemáticas y módulos CMFO.
